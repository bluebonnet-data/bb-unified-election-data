# Contributing to this project

Thank you for volunteering your time. This document explains how we work.

## Our philosophy

This is a 100% volunteer project. We respect your time, your day job, and 
your boundaries. Everything is designed so you can contribute meaningfully 
in 20-minute windows without needing to attend meetings or make big commitments.

## How to pick up a task

1. Go to the [Issues tab](../../issues)
2. Find an issue that matches your skills — look for the "good first issue" label if you are new
3. Leave a comment on the issue saying you are working on it
4. When you are done, open a Pull Request and reference the issue number (e.g. "Closes #2")

## The Handover Rule

If you need to pause your work before it is finished, leave a comment on your 
active issue or branch that explains:

- What you completed
- What is left to do
- Any gotchas or blockers you ran into

This lets another volunteer pick up exactly where you left off without needing 
to track you down.

## No unnecessary meetings

We do not meet to talk about work. All collaboration happens through GitHub 
Issues and Pull Request comments. If you have a question, post it as a comment 
on the relevant issue — that way the answer is visible to everyone.
