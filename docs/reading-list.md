# References & Reading List

This document contains the core research, methodology, and legal context for this project.

## 🏛️ Civic Data & Redistricting Context
* [Redistricting Data Hub](https://redistrictingdatahub.org/) – Our primary reference for national data standards and guides.
* [Redistricting Resources](https://redistrictingdatahub.org/resources/) – Explainers on aggregation and spatial data challenges.
* [Data for Democracy: The Basics](https://data-democracy.org/basics) – Foundational concepts on the intersection of data and representation.

## 🛰️ Technical Methodology & Geometry
* [Political Geometry: Hachadoorian & Buck](https://data-democracy.org/publications/political-geometry/13-HachadoorianBuck.pdf) – **Core Methodology.** This paper outlines the geometric difficulties of mapping results across changing boundaries.
* [Census Block Fundamentals](https://www.census.gov/geographies/mapping-files/time-series/geo/tiger-line-file.html) – Information on the "atomic unit" of our spatial joins.

## 📦 Data Sources Used in This Project

* [Texas Legislative Council Capitol Data Portal](https://data.capitol.texas.gov/) — Source for precinct boundaries, congressional district shapefiles, and election analysis files.
* [2020 Census P.L. 94-171 Redistricting File](https://www2.census.gov/programs-surveys/decennial/2020/data/01-Redistricting_File--PL_94-171/Texas/) — Census block population data used for population weighting.
* [VEST 2020 Texas Precinct Election Results](https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/K7760H) — Precinct-level 2020 Presidential results via Harvard Dataverse.
* [Census TIGER/Line Shapefiles](https://www.census.gov/geographies/mapping-files/time-series/geo/tiger-line-file.html) — Census block geometry files.
